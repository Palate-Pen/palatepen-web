Founder Admin v1 — Design Notes
Five domains, one command centre. Sidebar splits into Overview (Home) and Domains (Users & Kitchens, Business, System Health, Content & Comms, Founder Ops). Each domain is its own full page; Home is the dashboard view that surfaces what matters now across all five.

Authentication: locked to jack@palateandpen.co.uk via Supabase auth. The page checks the current Supabase session and rejects anyone whose email doesn't match. No hardcoded password, no separate auth system — your real Supabase login gets you in. Sign out clears the Supabase session and redirects to the public site.

Looking Ahead applies here too. Forward-intelligence detectors run against business + user data: free kitchens hitting limits (conversion signal), active kitchens dropping usage (churn risk), Stripe payment failures (revenue risk), feature flag rollouts ready to ship (release ready). Same pattern as chef shell — the system surfaces signals, you decide what to do.

Recent sign-ups table is the v8 status-led row pattern with tier pills replacing status pills. Free / Kitchen / Pro tier pills carry visual hierarchy — Pro fills gold, Kitchen gold-tinted background, Free muted. Activity dots show engagement at a glance (healthy / dormant / churned). Click any row to drill into that user's full data + admin actions (impersonate, change tier, suspend, message).

Quick actions surface frequent flows. Find a user / send announcement / update seasonal calendar / toggle feature flag — these are the four highest-frequency founder operations. Each lands on the relevant domain page pre-scoped.

Two-column panels handle the rest. Business Snapshot (this month vs last) and System Health (infra status) on the first row. Content & Comms and Founder Ops on the second. Each panel is glanceable in 5 seconds, deep-divable on click.

The voice stays consistent with the chef shell. Day-not-time, casual-name first, italic gold accents, no SaaS jargon. "Twelve kitchens. Four on Pro. £294 MRR. One issue needs your eye." reads like a founder talking to themselves — direct, observational, no corporate fluff.
