Menus v1 — Live Menu Reference
Menus is the chef's live reference. Three columns (Starters / Mains / Desserts) show today's menu with each dish's cost per unit and portion count. Chef can click through to the full recipe in the Recipes tab, or hit "Reference Recipe" to see technique inline. Edit Menu button lets the chef adjust (swap a dish, add a seasonal alternative). Print button exports a kitchen copy + front-of-house copy.

Cost transparency. Total menu cost per cover (£22.34), total for tonight's covers (£3,171.88), and margin health (68% = healthy green). If the menu drifts expensive (cost spikes because of ingredient price moves in The Bank), the chef sees it instantly.

Cross-surface. Clicking a dish name opens the recipe detail. Clicking cost takes you to The Bank filtered by that ingredient. Clicking "Edit Menu" opens the manager's menu builder for full menu revision (future: manager access only, or chef access with approval workflow).
